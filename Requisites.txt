##robomechtrix ##Simulate_Your_Robot_Arm_In_ROS_Noetic

Here I have used:

>> Ubuntu 20.04 (Focal Fossa)
>> ROS Noetic Ninjemys
>> Gazebo 11
>> MoveIt 1

Command to install the requisites:

1) sudo apt update
   sudo apt install ros-noetic-desktop-full

2) sudo apt update
   sudo apt install gazebo11 ros-noetic-gazebo-ros-pkgs ros-noetic-gazebo-ros-control

3) sudo apt update
   sudo apt install ros-noetic-moveit

4) sudo apt update
   sudo apt install ros-noetic-rviz

5) sudo apt install ros-noetic-ros-control ros-noetic-ros-controllers
